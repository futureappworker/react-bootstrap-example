# react-bootstrap-example

React example based on next.js.

## technology stack

- react
- typescript
- bootstrap
- scss
- eslint ( airbnb )


## install

```bash
npm install
```

## run the development server

```bash
npm run dev
```

## run the production server

```bash
npm run build
npm run start
```

## run eslint

```bash
npm run lint
```
